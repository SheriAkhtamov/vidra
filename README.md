# 🎬 Vidra — Видеозагрузчик (1000+ сайтов)

<p align="center">
  <img src="vidra_logo.png" alt="Vidra Logo" width="120">
</p>

**Vidra** — настольное приложение с красивым графическим интерфейсом для скачивания видео и плейлистов с **1000+ сайтов**.

Поддерживаемые платформы: **YouTube** • **RuTube** • **TikTok** • **Vimeo** • **VK** • и все остальные сайты, поддерживаемые [yt-dlp](https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md).

---

## ✨ Возможности

- 🔗 Скачивание видео по ссылке
- 📁 Скачивание целых плейлистов
- 🎚 Выбор качества: **4K** / **1080p** / **720p** / **480p** / **360p**
- 🎵 Скачивание **только аудио** (MP3)
- 📝 Скачивание субтитров
- 📋 Очередь загрузок — добавляй несколько задач и запускай одним кликом
- 📜 История скачиваний
- ⚡ Ограничение скорости скачивания
- 🏷 Встраивание метаданных и обложек (через FFmpeg)
- 📄 Настраиваемые шаблоны имён файлов

---

## 📋 Требования

| Зависимость | Версия | Обязательно |
|---|---|---|
| Python | 3.10+ | ✅ |
| yt-dlp | последняя | ✅ |
| FFmpeg | последняя | ⚠️ рекомендуется |
| customtkinter | — | ✅ |
| Pillow | — | ✅ |

> [!NOTE]
> Без FFmpeg видео будет скачано, но без склейки видео/аудио дорожек и без встраивания метаданных.

## 🚀 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourname/vidra
cd vidra
```

### 2. Создание виртуального окружения

**Windows:**
```bash
py -m venv .venv
.venv\Scripts\activate
```

**Linux / macOS:**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -U pip
pip install customtkinter pillow yt-dlp
```

---

## 🔧 Установка FFmpeg

### Windows (автоматически)

```bash
python download_ffmpeg.py
```

Скрипт скачает FFmpeg и сохранит его как `ffmpeg_bundled.exe` рядом с приложением.

### Linux

```bash
sudo apt install ffmpeg
```

### macOS

```bash
brew install ffmpeg
```

## ▶️ Запуск

```bash
python app.py
```

## 📖 Как пользоваться

1. Откройте вкладку **Скачать**
2. Вставьте ссылку на видео или плейлист
3. Нажмите **🔍 Найти форматы**
4. Выберите нужное качество (пресет или конкретный формат)
5. Нажмите **⬇ Скачать** — или добавьте в **очередь** кнопкой **+ В очередь**

> [!TIP]
> Можно добавить несколько видео/плейлистов в очередь и скачать их все разом во вкладке **Очередь**.

---

## 📦 Сборка в `.exe` (Windows)

Просто запустите `BUILD.bat` — он сделает всё сам:

```
BUILD.bat
```

Скрипт автоматически:
1. Проверит наличие Python
2. Установит все зависимости (`customtkinter`, `yt-dlp`, `pyinstaller`, `pillow`)
3. Скачает `yt-dlp.exe` и `ffmpeg.exe` для встраивания в сборку
4. Соберёт приложение через PyInstaller
5. Откроет папку `dist/` с готовым `Vidra.exe`

---

## ❓ Возможные ошибки

### `yt-dlp` не найден

```bash
pip install -U yt-dlp
```

Или поместите файл `yt-dlp.exe` рядом с приложением.

### FFmpeg не найден

Установите FFmpeg или запустите:

```bash
python download_ffmpeg.py
```
