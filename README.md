# Vidra - Video Downloader (1000+ сайтов)

Vidra - это настольное приложение с графическим интерфейсом, которое
позволяет скачивать видео и плейлисты с более чем 1000 сайтов.
Приложение использует **yt-dlp** как движок скачивания и **FFmpeg** для
обработки видео.

Поддерживаемые сайты включают: - YouTube - RuTube - Vimeo - TikTok -
VK - и многие другие (все сайты, поддерживаемые yt-dlp)

------------------------------------------------------------------------

# Возможности

-   Скачивание видео по ссылке
-   Поддержка плейлистов
-   Выбор качества (4K, 1080p, 720p, 480p, 360p)
-   Скачивание только аудио (MP3)
-   Скачивание субтитров
-   Очередь загрузок
-   История скачиваний
-   Ограничение скорости скачивания
-   Встраивание метаданных и обложек (через FFmpeg)

------------------------------------------------------------------------

# Требования

Перед запуском убедитесь, что у вас установлены:

-   Python 3.10+
-   yt-dlp
-   FFmpeg (рекомендуется)
-   Python библиотеки:
    -   customtkinter
    -   pillow

------------------------------------------------------------------------

# Установка

## 1. Клонирование репозитория

git clone https://github.com/yourname/vidra cd vidra

## 2. Создание виртуального окружения

Windows:

py -m venv .venv .venv`\Scripts`{=tex}`\activate`{=tex}

Linux / macOS:

python3 -m venv .venv source .venv/bin/activate

## 3. Установка зависимостей

pip install -U pip pip install customtkinter pillow yt-dlp

------------------------------------------------------------------------

# Установка FFmpeg

### Windows (автоматически)

python download_ffmpeg.py

Скрипт скачает FFmpeg и сохранит его как:

ffmpeg_bundled.exe

### Linux

sudo apt install ffmpeg

### macOS

brew install ffmpeg

------------------------------------------------------------------------

# Запуск приложения

python app.py

После запуска откроется графический интерфейс программы.

------------------------------------------------------------------------

# Использование

1.  Откройте вкладку **Скачать**
2.  Вставьте ссылку на видео
3.  Нажмите **Найти форматы**
4.  Выберите качество
5.  Нажмите **Скачать**

Можно также добавить несколько задач в **очередь** и скачать их
одновременно.

------------------------------------------------------------------------

# Сборка в .exe (Windows)

Установите PyInstaller:

pip install pyinstaller

Соберите приложение:

pyinstaller --onefile --noconsole app.py --icon vidra.ico

После сборки .exe появится в папке:

dist/

------------------------------------------------------------------------

# Возможные ошибки

## yt-dlp не найден

Установите:

pip install -U yt-dlp

или поместите файл **yt-dlp.exe** рядом с приложением.

## FFmpeg не найден

Установите FFmpeg или запустите:

python download_ffmpeg.py
